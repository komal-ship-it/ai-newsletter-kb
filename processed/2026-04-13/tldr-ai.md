## Techniques & Methods
- **Agentic Engine Optimization (AEO)** — Structuring documentation so AI coding agents can consume it effectively. Key dimensions: discoverability, parsability, token efficiency, capability signaling, access control (robots.txt). Sites that ignore AEO cause agents to hallucinate or fail tasks.
- **Agent-Driven Cloud Operations** — Cloud-hosted coding agents triggered by alerts or schedules, running in isolated sandboxes without human in the loop. Pattern for automating SRE toil (incident response, routine ops) while reserving humans for SLO design and capacity planning.
- **Thin Harness, Fat Skills** — Architecture pattern: Markdown procedures (fat skills) encoding judgment/domain knowledge sit on top; thin CLI harness in the middle; deterministic foundation at bottom.
- **Open Memory Harnesses** — Agent memory should be user-owned and open so teams can build proprietary datasets they control.
