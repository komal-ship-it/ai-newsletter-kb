## 🛒 Shopify AI Toolkit — Your Store Controlled by Plain English

Shopify open-sourced a free plugin on April 9 that connects AI coding agents directly to your store. Point Claude Code, Cursor, or VS Code at it, and you can update products, manage inventory, run bulk SEO changes, and execute pricing rules in plain English. No dashboard login needed.

How It Works: Builds on [Shopify's Dev MCP Server](https://shopify.dev/docs/apps/build/ai-toolkit) — connects to Claude Code, Cursor, VS Code, Gemini CLI, and OpenAI Codex. The 16 skills cover: learning the Shopify API surface, pulling live documentation, validating code, and executing store operations.

Free and MIT-licensed. Standard API rate limit: 1,000 cost points per minute.
[Try Shopify CLI](https://shopify.dev/docs/apps/build/ai-toolkit)

## 📄 Confluence Remix — Turn Any Doc Into a Visual in Seconds

Atlassian launched Remix in open beta on April 8 for Confluence Cloud users with Rovo. Highlight any section of a Confluence page and Remix generates a chart, infographic, or scorecard layered on top. The original content stays intact and updates automatically.

Three partner agents launched April 13:
- [Lovable](https://www.atlassian.com/software/confluence): turns a product spec into a working interactive prototype
- Replit: converts a technical document into a starter app you can fork
- Gamma: transforms meeting notes into a polished presentation

[Try Atlassian Remix](https://support.atlassian.com/confluence-cloud/docs/transform-content-with-remix/)

## 🤖 Claude Managed Agents — Ship Cloud Agents Without the Infrastructure Work

Anthropic launched Claude Managed Agents in public beta on April 8. You define the task, Anthropic runs the infrastructure — sandboxed code execution, state management, context compaction, and end-to-end tracing.

Early adopters: Notion, Rakuten, and Asana. Available to all Claude API accounts today.

Built-in Tools:
- Bash — run shell commands in the container
- File operations — read, write, edit, glob, grep
- Web search and fetch — live research
- [MCP server](https://modelcontextprotocol.io/introduction) connections

Pricing: Standard Claude token rates plus $0.08 per session-hour for active runtime.
[Try Claude Managed Agents](https://claude.com/form/claude-managed-agents)
